# CosmoCats

Side-scroller game with a level editor.

 * Run the game from **TitleScreen** scene.
 * Run the level editor from **Editor** scene.
 
 
 ## Dependencies:
 * StylizedExplosionPack1
 * Tween for Unity by Jeff Johnson
